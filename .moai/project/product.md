# Snake Game - 제품 정의서

## 프로젝트 개요

**프로젝트명**: Snake Game
**타입**: 웹 애플리케이션 (브라우저 기반 게임)
**개발 언어**: TypeScript
**빌드 도구**: Vite
**타겟 플랫폼**: 현대 웹 브라우저 (Chrome, Firefox, Safari, Edge)
**렌더링**: HTML5 Canvas API

---

## 제품 설명

웹 브라우저에서 즉시 구동할 수 있는 **클래식 스네이크 게임**입니다. 별도의 설치 없이 모던 브라우저에서 게임을 플레이할 수 있으며, 점수 시스템과 최고 기록 관리, 점진적인 난이도 상향을 통해 중독적인 게임플레이를 제공합니다.

### 핵심 가치 제안

- **즉시 접근성**: 브라우저에서 별도 설치 없이 바로 플레이
- **클래식 경험**: 향수를 불러일으키는 원본 스네이크 게임의 매력
- **진행 동기**: 최고 점수 달성을 통한 반복 플레이 유도
- **점진적 난이도**: 레벨 시스템을 통한 자연스러운 난이도 상향

## 핵심 기능

### 1. 클래식 게임플레이

#### 게임 메커닉스
- **방향 조작**: 화살표 키로 스네이크 방향 제어 (상, 하, 좌, 우)
- **성장 시스템**: 먹이를 먹을 때마다 스네이크 길이 증가
- **게임 오버 조건**:
  - 벽에 충돌 (게임 보드 경계 넘어감)
  - 자신의 몸과 충돌 (뱀이 자기 몸과 만남)

#### 게임 흐름
1. 게임 시작 시 스네이크는 중앙에서 초기 길이로 배치
2. 플레이어는 화살표 키로 방향 제어
3. 스네이크는 매 프레임 자동으로 이동
4. 먹이와 충돌 시 점수 증가 및 몸 길이 증가
5. 장애물 충돌 시 게임 오버

### 2. 점수 시스템 및 최고 기록

#### 점수 관리
- **현재 점수**: 게임 진행 중 실시간 표시
- **점수 계산**: 먹이 1개 = 10 포인트
- **누적 점수**: 게임 진행 중 계속 누적

#### 최고 기록 관리
- **localStorage 저장**: 최고 점수를 브라우저 로컬 스토리지에 영구 저장
- **자동 업데이트**: 새로운 최고 기록 달성 시 자동 저장
- **게임 시작 시 로드**: 게임 시작 시 이전 최고 기록 불러오기
- **UI 표시**: 현재 게임 점수와 최고 기록 동시 표시

### 3. 레벨 및 속도 시스템

#### 레벨 진행
- **레벨 진행 조건**: 먹이 4개를 먹을 때마다 레벨 1 상향
- **시각적 피드백**: UI에서 현재 레벨 표시
- **최대 레벨**: 제한 없음 (플레이어 실력에 따라 계속 진행 가능)

#### 속도 증가
- **기본 속도**: 레벨 1에서 적정 속도로 시작
- **속도 계산 공식**: 게임 속도 = 기본 속도 × (1 + 레벨 × 0.15)
  - 레벨 1: 기본 속도
  - 레벨 2: 기본 속도 × 1.15
  - 레벨 3: 기본 속도 × 1.30
  - 레벨 4: 기본 속도 × 1.45

#### 게임 균형
- 속도 증가로 게임 난이도 점진적 상향
- 플레이어 반응 시간과 게임 속도의 균형 유지
- 무한 진행 가능성으로 높은 점수 달성 동기 제공

### 4. 시각 효과 및 사용자 경험

#### 애니메이션
- **스네이크 이동 애니메이션**: 부드러운 이동 효과
- **먹이 획득 애니메이션**: 먹이 먹을 때 화면 진동 또는 플래시 효과
- **게임 오버 애니메이션**: 스네이크와 게임 보드 페이드 아웃 효과
- **레벨 업 애니메이션**: 레벨 표시 화면 확대/축소 효과

#### 사운드 효과
- **먹이 획득 음**: 게임 점수 획득 시 효과음
- **게임 오버 음**: 게임 종료 시 비프음
- **배경음**: 게임 진행 중 반복되는 배경 음악 (선택사항)
- **음소거 버튼**: 사용자가 음소거 옵션 선택 가능

#### 아름다운 UI/UX 디자인
- **색상 배치**: 대비가 명확한 색상 조합으로 시각성 향상
  - 배경: 어두운 톤 (진정 효과)
  - 스네이크: 선명한 그린 또는 네온 그린
  - 먹이: 붉은색 또는 화려한 색상
  - UI 요소: 명확한 흰색 텍스트

- **UI 요소**:
  - 게임 제목 및 로고
  - 현재 점수 및 최고 기록 패널
  - 현재 레벨 표시
  - 게임 시작/다시 시작 버튼
  - 일시정지/재개 버튼

- **반응형 디자인**:
  - 모바일, 태블릿, 데스크톱 모두 지원
  - 터치 조작 지원 (모바일)
  - 화면 크기에 맞춘 게임 보드 스케일 조정

- **접근성**:
  - 명확한 버튼 레이블
  - 충분한 색상 대비 (WCAG 2.1 준수)
  - 키보드 네비게이션 지원

## 사용 시나리오

### 시나리오 1: 신규 플레이어 게임 시작

1. 사용자가 게임 URL에 접속
2. 로딩 완료 후 게임 화면 표시
3. "게임 시작" 버튼 클릭
4. 게임 보드 중앙에서 스네이크와 먹이가 나타남
5. 화살표 키로 스네이크 조작 시작
6. 게임 진행 (점수, 레벨, 속도 표시)
7. 충돌 시 게임 오버 및 최종 점수 표시
8. "다시 시작" 버튼으로 새 게임 시작

### 시나리오 2: 최고 기록 도전

1. 이전 플레이 기록 존재 시 "최고 기록: XXX" 표시
2. 플레이어가 해당 점수를 초과하기 위해 게임 플레이
3. 새로운 최고 기록 달성 시 축하 메시지와 함께 저장
4. 브라우저 방문 시 새로운 최고 기록 유지 표시

### 시나리오 3: 점진적 난이도 도전

1. 게임 시작 시 레벨 1, 기본 속도
2. 먹이 4개 획득 후 레벨 2로 상향 (속도 1.15배)
3. 계속 플레이하면서 레벨 상향 (속도 계속 증가)
4. 높은 레벨에서의 도전적인 게임플레이
5. 개인 최고 레벨 달성 기록
